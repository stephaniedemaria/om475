Project 2: SAS EM Project

I completed this project in OMIS 482: Predictive Analytics.
For this project, I had to work with a group to find a data set
and use SAS Enterprise Miner to perform predictive analytics.
We did our project using data from the olympics covering athlete and
medal information.

We used SAS EM to build multiple decision trees and run a few regression
nodes. The model that we built is displayed to the left. In the end,
we did not find a ton of interesting information from this data, but I
did learn a lot about using SAS EM.
